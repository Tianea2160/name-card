# NAME CARD

나만의 작은 이름표 만들기 프로젝트입니다.

동아리 박람회의 체험형 컨텐츠를 목적으로 제작했습니다.

자신의 이름을 입력하면 이름이 들어간 500x500 사이즈의 name card가 만들어지고 해당 카드를 다운로드 할 수 있습니다.

- canvas 기능을 이용해서 사용자의 동적으로 이미지를 수정하여 사용자 name card 생성
